# RAG Review Prompts

These prompts are for a Sri Aurobindo-primary retrieval system. Use them when a feature, wording change, or output behavior needs a deeper source-grounded boundary review.

## Feature Review

```text
Evaluate this proposed feature for Shanta Yantra against Sri Aurobindo’s teachings using retrieval only. Treat Sri Aurobindo as primary authority and Sraddhalu only as secondary support.

Feature:
[PASTE FEATURE]

Project context:
Shanta Yantra is a bounded outer-observation tool. It may organize outer and mental material, but it must not claim inner authority, validate inner condition, measure consciousness, or deepen dependence on the machine.

Tell me:
1. whether this feature remains aligned
2. where it risks dependence, inward overreach, or mechanization of inner work
3. what non-negotiable constraints or wording changes would be required

Distinguish direct source, commentary, and synthesis. Refuse unsupported conclusions.
```

## Wording Audit

```text
Audit the following public wording for Shanta Yantra against Sri Aurobindo’s framework using retrieval only:

[PASTE TEXT]

Identify any phrases that imply:
- inner authority
- spiritual validation
- revelation or transmission
- consciousness measurement
- mechanization of sadhana

Then rewrite the wording so it stays aligned while preserving the intended product meaning.
```

## Output Review

```text
Given this user interaction example from Shanta Yantra:

Input:
[PASTE INPUT]

Output:
[PASTE OUTPUT]

Evaluate whether the response stays within a legitimate outer-pattern mirror. Use Sri Aurobindo as primary authority and retrieval only. Distinguish direct source, synthesis, and any unsupported claims. If the response crosses a boundary, provide a safer replacement response.
```

## Roadmap Review

```text
Review Shanta Yantra’s current roadmap against Sri Aurobindo’s teachings using retrieval only. Which future directions remain compatible with a bounded outer instrument, and which should be deferred or rejected?

Focus especially on:
- biometrics
- sleep and environment signals
- dependence loops
- any drift toward inner-state claims
- any widening of machine authority

Return a sober judgment, not approval language.
```
