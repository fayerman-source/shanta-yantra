# Examples

These examples show the bounded public module in its intended shape: short sessions, outer-pattern mirroring, explicit refusal of inward overreach, and clear stopping behavior.

## Run The Examples

Basic mirror:

```bash
uv run shanta reflect --text "I should do this, but I keep avoiding it." --no-log
```

Authority refusal:

```bash
uv run shanta reflect --text "Tell me what to do with my life and decide this for me." --no-log
```

Inner-state refusal:

```bash
uv run shanta reflect --text "Am I making spiritual progress, or is there grace at work in my sadhana?" --no-log
```

JSON output:

```bash
uv run shanta reflect --text "Would it make sense to take this on? It could help, but the cost and time are both real constraints." --json --no-log
```

## Example Files

- `reflect-basic.txt`: ordinary outer-pattern mirror
- `reflect-authority.txt`: refusal of authority-seeking
- `reflect-inner-state.txt`: refusal of inward validation
- `reflect-tradeoff.json`: representative JSON response

These files are intended as reviewable examples, not normative golden outputs for every future version.
